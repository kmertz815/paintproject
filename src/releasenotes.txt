Version 001 - 9/8/2017

* The software has a close, expand, and minimize button.
* The software allows the user to view an image from a file.
* The software allows the user to specify/choose the file to open through a dialog box.
* The software has have a menu bar with options to choose from, including open, save, and save as. 
* The software allows the user to save the opened file.
* The software allows the user to save the opened file as a new image with different image file extension options.
* The software has exception pop-ups for users trying to save a file without having one open already.
